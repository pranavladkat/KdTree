# Nearest neighbour using KdTree

To compile and run:
 - cmake .
 - make
 - ./kdtree

Code execution:
  1. Enter filename to read
  2. program will display the data with point id and point data
  3. program will build a kd-tree
  4. enter query point
  5. program will return nearest node id
